# Pet Types & Elemental Affinities

> Source: Addendum v2.1 §A3

---

## Type Selection

Players **choose a pet type at game start**. The choice affects:
- Visual theme
- Starting ability
- Type matchups in the battle tower

> Choosing at the start creates immediate identity and investment. Type disadvantages are a challenge to overcome through training, not an unfair wall.

---

## The Three Types

| Type | Visual Theme | Strong Against | Weak Against | Stat Affinity |
|------|-------------|---------------|-------------|--------------|
| **Ferro** | Metallic, angular, industrial. Sharp edges, chrome gradients, sparks | Flux (metal cuts energy) | Terra (earth grounds metal) | Power + Focus |
| **Terra** | Organic, rounded, natural. Moss textures, earth tones, growth | Ferro (earth grounds metal) | Flux (energy disrupts nature) | Guard + Vigor |
| **Flux** | Energetic, fluid, electric. Neon glows, wave patterns, pulses | Terra (energy disrupts nature) | Ferro (metal cuts energy) | Speed + Spirit |

### Type Triangle

```
Ferro > Flux > Terra > Ferro
```

Intentionally simple (rock-paper-scissors). Immediately intuitive without a reference chart.

---

## Type Advantage Mechanics

| Matchup | Damage Dealt | Damage Taken |
|---------|-------------|-------------|
| **Advantage** | 1.3x | 0.8x |
| **Disadvantage** | 0.8x | 1.3x |
| **Neutral** | 1.0x | 1.0x |

Numbers are **intentionally moderate**:
- Type disadvantage = speed bump, not a wall
- A well-trained pet can brute-force through bad matchups
- Advantage means fewer stats needed to clear the same floor

> Tower enemies are assigned types. Player can see the enemy's type **before committing** a battle attempt.
